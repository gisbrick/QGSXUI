# Bienvenido a QGSXUI

Esta es la librería de componentes frontend para aplicaciones QGIS.

## Componentes disponibles

Puedes explorar los componentes disponibles en el menú lateral. Los componentes están organizados en las siguientes categorías:

- **UI**: Componentes de interfaz de usuario básicos (Button, Modal, Message, etc.)
- **QGS**: Componentes específicos para QGIS (Map, Form, Table, etc.)
- **UI_QGS**: Componentes UI adaptados para QGIS

## Uso

Cada componente incluye ejemplos de uso y documentación. Puedes copiar el código de los ejemplos para usarlo en tu aplicación.
